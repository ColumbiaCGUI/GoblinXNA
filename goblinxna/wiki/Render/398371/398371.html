CUSTOM MARKER LAYOUTS USING THE MarkerLayout TOOL

By running this GoblinXNA utility, you can create your own arrays of fiducials for you unique needs.

First, navigate to the MarkerLayout directory. On my computer it is found at "C:\Users\Colin\Documents\GoblinXNAv3.5\tools\MarkerLayout". Then open the "MarkerLayout.sln" file.

You can click through any subsequent warnings about the location not being trusted.

Once the solution is open, build and run the project by navigating the menu system or hitting the F5 key. This will produce a folder at bin\Debug that contains a new marker layout produced by the SampleProgram that is provided with the solution.
